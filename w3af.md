#w3af
[http://w3af.org/](http://w3af.org/)
w3af is a Web Application Attack and Audit Framework. The projectâ€™s goal is to create a framework to help you secure your web applications by finding and exploiting all web application vulnerabilities.

It works in 3 steps
1. Identify all links, forms, query string parameters.
2. Send specially crafted strings to each input and analyze the output
3. Generate a report with the findings
